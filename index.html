git 사용법 1. git init으로 master branch를 생성 2. git status로 현재 상태 확인 3. git add . :현재 경로에 있는 모든 수정 사항들을 스테이지에 올려 새로운 버젼 생성 4. git commit -m 'message' : commit과 버젼 생성 5. git log로 저장되었는지 확인 --new branch-- 1. git branch add-style: add-style이라는 new branch 생성 2. git branch: branch list출력 3. git checkout add-style: add-style로 branch 이동 4. git branch -d remove-this: remove-this라는 branch 삭제 +현재 사용중인 브랜치는 삭제x 5. git checkout -b abc: abc라는 branch 생성과 동시에 이동 --git merge-- 병합 1. git merge add-style: 현재 branch에 add-style을 병합 -> 병합시 기존 파일과 충돌이 일어날 수 있음. --git reset-- 삭제, 버전 되돌리기 1. git reset --hard HEAD~: 가장 최근 COMMIT삭제 2. git reset --hard ORIG_HEAD: 가장 최근 삭제된 COMMIT복구하며 돌아감. 3. git reset --hard HEAD~3 : 총 세 단계 뒤로 돌아감. 4. git reset --hard a4a802ecb2f587415584280542541c6f3ea08be6: 돌아갈 버젼의 commit아이디를 입력하여 복구 --github upload-- 1. git remote add origin https://github.com/antipiebse/test-repo.git origin은 저장할 별칭 2. git push origin master: master branch만 저장 3. git push origin --all: 모든 branch를 저장 --git ignore-- 업로드하고 싶지 않은 파일 지정 a.txt > .gitignore --git diff-- 변경사항을 알아볼때 사용 --git clone-- git clone 복사할 저장소 git clone --branch [TAG] [REPO_URL] : 원하는 태그나 브랜치 복사 --git 파일 가져오기-- git fetch로 변경사항 확인하기 git pull origin master 가져오기
